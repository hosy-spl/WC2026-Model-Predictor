# FIFA World Cup 2026 Predictions and Analysis

## Overview
This project predicts the 2026 FIFA World Cup tournament using Machine Learning models built in R. The analysis was conducted by applying concepts learnt from my university's Year 1 introductory module in Data Science.

## Objectives
1. Identify competitive group stage matches worth watching
2. Predict the full tournament bracket
3. Post-tournament analysis

## Models Used
- **Logistic Regression**
- **Linear Regression**

## Data Sources
- Historical World Cup matches (1930-2022) from Kaggle
- 2026 World Cup fixtures from Kaggle
- FIFA Rankings from Wikipedia
- Tournament Brack Simulation from The Straits Times

## Tools
- R
- Libraries: `nnet`
